# Prometheus Camera

Prometheus Camera 是面向 Xiaomi 14 Ultra（`aurora`）的 Xiaomi 17 Ultra 相机移植项目。

本仓库用于发布可安装的二进制构建，不提供工程源码。请仅从本仓库的 [Releases](https://github.com/benbaobaoshigemi/Prometheus-Camera/releases) 页面获取正式版本。

## 适用范围

- 目标设备：Xiaomi 14 Ultra（`aurora`）
- 系统环境：已取得 Root 权限，并具备安装修改版系统应用所需的核心破解环境
- 发布内容：相机 APK、配套模块及安装说明

其他设备、系统版本或安装环境未经验证，不建议尝试。安装前请自行备份重要数据，并确保具备恢复系统的能力。

## 下载与安装

前往 [最新版本](https://github.com/benbaobaoshigemi/Prometheus-Camera/releases/latest) 下载完整组合包，并严格按照压缩包内的安装说明操作。

请勿混用不同版本的 APK 与模块。升级前请阅读对应版本的 Release Notes；如需反馈问题，请在 [Issues](https://github.com/benbaobaoshigemi/Prometheus-Camera/issues) 中附上设备系统版本、项目版本、复现步骤和必要日志。
